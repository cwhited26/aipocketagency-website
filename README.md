# aipocketagency-website

Marketing landing page for **AI Pocket Agency** — `aipocketagency.com`.

Single-page Next.js 14 (App Router) site with the locked Skool sales copy and a single "Join the Founding 50" CTA pointing to the Skool group.

## Stack

- Next.js 14 (App Router, TypeScript strict)
- Tailwind CSS
- Static / no DB, no auth, no API routes
- Deployed on Vercel

## Local dev

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
```
